# NewsBytes
## Install Django
- Install [Django](https://docs.djangoproject.com/en/2.0/topics/install/) from here.
## All DataBase Create 
run command 
```sh
python manage.py migrate
```
## Create Admin
```sh
python manage.py createsuperuser
```
and follow instructions
## Run Server
```sh
python manage.py runserver
```
